Language Translator

Translate text between multiple languages instantly!

What is it?

This web application allows you to translate text from one language to another using advanced machine translation technology. It's fast, easy to use, and supports a wide range of languages.

How was it made?

Built with Python and Flask for the backend, and Google Translate API for language conversion. The frontend is designed for a vibrant, modern look using HTML and CSS. The project demonstrates RESTful API design and seamless integration between frontend and backend.

Schema Overview

  • Frontend: index.html, /translate page
  • Backend: Flask app (app.py), translation logic
  • API Endpoint: /translate (POST: text, source_lang, target_lang)
  • Response: Translated text in JSON format
